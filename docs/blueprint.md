# **App Name**: Sofol IT

## Core Features:

- Dynamic Hero Narrative: Features an animated headline ('Idea to Product'), a 'Luminous Beam' background effect, and 'Magnetic' CTA buttons with glowing hovers, establishing the brand's core value proposition instantly.
- Guided Process Bento Grid: A 4-column Bento grid showcasing the 'Concept → Strategy → Design → Code' journey with subtle hover-glows and 'Progress' line animations connecting each stage.
- Interactive Portfolio Showcase: Displays past work using 'Device Mockup' components, featuring either a horizontal scroll or 'Parallax' stacking cards to engage visitors.
- Automated Social Proof Integration: Incorporates an 'Infinite Marquee' of tech stacks and partner logos, alongside 'Glass' effect testimonial cards with animated border-beams to build trust.
- High-Fidelity Contact & Conversion Form: A minimalist, premium contact form integrated with react-hook-form and zod for seamless and frictionless lead generation.
- AI-Powered Project Ideation Tool: An interactive 'Project Calculator' or 'Idea Starter' tool that provides initial project insights and suggestions based on user input, engaging potential leads immediately.

## Style Guidelines:

- Base background color is a deep, luxurious dark (#030303), providing a high-contrast canvas for vibrant accents and creating a premium aesthetic.
- Primary accent color: A striking electric blue (`#3396E6`). This vibrant hue evokes technology and innovation, offering strong visual impact against the dark background. (HSL: H=220, S=80%, L=50%)
- Complementary accent color: A luminous cyan (`#2ED0E6`). Used for highlights, interactive elements, and supporting gradient effects, creating a dynamic visual flow. (HSL: H=190, S=80%, L=55%)
- Strategic use of 'Mesh Gradients' for atmospheric accents, featuring blends like Indigo to Violet or Electric Blue to Cyan to add depth and modern flair to specific UI elements.
- The 'Inter' sans-serif typeface is recommended for all text elements. Its modern, clean aesthetics and excellent legibility make it ideal for both massive, bold headings and readable body content.
- The latest 'Lucide' icon set will be utilized for crisp, modern visual cues and to ensure consistent iconography throughout the application.
- A dynamic and subtle grid background pattern (implemented via SVG or CSS `background-size`) with a radial mask that gently fades toward the edges, creating a sense of depth and technical sophistication.
- Implements 'Glassmorphism' for headers and card backgrounds, employing high-blur backdrops to create a sophisticated sense of layered depth and focus on content.
- Smooth scroll-triggered entry animations (using Framer Motion or GSAP) will feature a 'y: 20, opacity: 0' to 'y: 0, opacity: 1' transition for all sections, providing an engaging content reveal.
- Directional slide-in animations, with 'Left-to-Right' movement for text and 'Right-to-Left' for supporting imagery, will be used to create dynamic movement and guide the user's eye.
- Interactive details like animated 'border-trace' effects for cards and glowing hover states for 'Magnetic' CTA buttons will enhance tactile feedback and user engagement, contributing to a premium feel.
